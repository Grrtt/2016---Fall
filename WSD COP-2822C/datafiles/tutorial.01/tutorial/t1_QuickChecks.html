Section 1.1

1. What is a markup language?

A markup language is a language that describes the content and structure of a document by identifying, or tagging, different elements in the document.

2. What is XHTML? How does XHTML differ from HTML?

XHTML stands for "Extensible Hypertext Markup Language." XHTML, unlike HTML, is very strict in its syntax.

3. What is the W3C? What is the WHATWG?

W3C is the World Wide Web Consortium, who have set up standards for Web browser manufacturers, and tried to push XHTML. WHATWG was an off-shoot of W3C that pushed forward development on HTML5, set to rival XHTML 2.0. Untimately, HTML5 won out as XHTML 2.0 was stalled indefinitely in 2009.

4. What is a doctype? What are two uses of the doctype?

A Doctype is a Document Type Declaration. Depending on the doctype used, the web page will render differently. Also, the document will be (in)validated depending on the doctype used and the document's following syntax.

5. What is incorrect about the syntax of the following code?

Welcome to the J-Prop Shop
The cose is written improperly, with the emphasis element straddling the closing paragraph tag. While it displays properly, its formatted in a less legible way than standard.

6. What is white space? How does HTML treat consecutive occurrences of white space?

White space is "empty" space in a document. HTML treats white space as inconsequential, and it will not be rendered.

7. What structural element would you use to mark a sidebar?

The aside tag.

8. What structural element would you use to mark the page footer?

The footer tag.

Section 1.2

1. Specify the code you would enter to mark the text The J-Prop Shop as an h1 heading and the text Product List as an h2 heading. Add code to group these two headings so browsers recognize them as a heading and subheading, respectively.

					<header>
						<hgroup>
							<h1>The J-Prop Shop</h1>
							<h2>Product List</h2>
						<hgroup/>
					</header>

2. Specify the code you would enter to mark the text Hamlet by William Shakespeare as an h1 heading, with a line break after the word Hamlet.

					<h1>Hamlet<br /></h1>

3. Create an ordered list of the following items: Packers, Bears, Lions, Vikings.

  1. Packers
  2. Bears
  3. Lions
  4. Vikings

4. Specify the code to access the CSS style sheet file uwstyles.css. Where should you place this code within an HTML file?

				<head>
					<link href="uwstyles.css" rel="stylesheet" type="text/css"/>
				<head/>

5. Mark the graphic file portrait.gif as an inline image, setting the dimensions to 250 pixels wide by 300 pixels high. Specify the text David Vinet as alternate text to be displayed in place of the image for non-graphical browsers.

<img src="portrait.gif" width="250" height="300" alt="David Vinet"/>

6. Specify the code to place the portrait.gif image from the previous question within a figure box with the caption David Vinet, owner of the J-Prop Shop.

				<figure>
					<img src="portrait.gif" width="250" height="300" alt="David Vinet"/>
					<figcaption>David Vinet<figcaption/>
				<figure/>
