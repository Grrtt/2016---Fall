Media Searcher

Welcome!

Welcome to my dynamic media search list. This site is designed to dynamically add Movies & TV Shows in a visible list that returns various information concerning the different media.

The site references various sites like IMDb, Rotten Tomatoes, and metacritic. The results are snappy, and are achieved through the OMDb API. Unfortunately, I was unable to find a free API that would return video clips for each media search. A sample of what it would have looked like can be found below.

Feel free to go to the Feedback section to send me any comments concerning the site.
